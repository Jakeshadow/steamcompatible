Forza Horizon 6 on Steam Machine

It doesn't run well — yet. Here's exactly what's broken, what's been fixed, and how to get the best experience possible right now.

Last updated: July 2, 2026 · Based on Proton #9789 (327+ reports)

Current Status: ⚠️ Playable with Major Issues

Forza Horizon 6 launched as a Steam Day 1 title, but Proton compatibility is rough. After 327+ community reports, the consensus is: you can play it on Steam Machine, but expect stuttering, and don't turn on ray tracing.

What's Broken (And Why)

1. Severe Stuttering — World Streaming

The most reported issue. FH6 streams open-world data aggressively, and Proton's translation layer introduces micro-stutters when loading new regions. Some stutters are visible in the frametime graph (30-50ms spikes), some are invisible — the game freezes for a fraction of a second without the graph registering it. This suggests a synchronization issue between the game's asset pipeline and Proton's D3D12→Vulkan translation.

What helps (community confirmed):

  • Set launch option: DXVK_ASYNC=1 %command% — allows async shader compilation, reducing stutter on first load
  • Reduce World Car Detail to Medium — fewer unique cars to stream in
  • Install on NVMe SSD — SATA SSDs amplify the streaming bottleneck

2. Ray Tracing — Instant GPU Crash

Turning ray tracing on (any setting) causes an immediate GPU hang. The error is a VK_ERROR_DEVICE_LOST — Proton's Vulkan layer loses communication with the GPU. This affects both AMD (RX 6000/7000/9000) and NVIDIA (RTX 3000/4000/5000) GPUs. Valve has not shipped a fix yet.

Workaround: Don't enable ray tracing. FH6's rasterized lighting is still excellent — the visual difference on Medium/High is subtle, and you're losing 30-40% FPS for a feature that crashes.

3. Black Screen on Startup — Fixed

Previously the game showed a black screen until you alt-tabbed out and back in. Fixed in Proton Experimental Bleeding Edge (build 11.0-362204, May 15 2026). If you're still seeing this, switch to Bleeding Edge: Proton Experimental → Properties → Betas → bleeding-edge.

How to Get the Best Experience Today

  1. Switch to Proton Experimental Bleeding Edge — fixes the black screen and includes latest D3D12 translation improvements
  2. Set launch options: DXVK_ASYNC=1 DXVK_CONFIG="dxvk.numCompilerThreads=4" %command%
  3. Cap to 1080p Medium — disable ray tracing, set World Car Detail to Medium, shadows to Low
  4. Use Gamescope for smoother frame pacing: gamescope -w 1920 -h 1080 -f -- %command%
  5. Install on NVMe — the world streaming bottleneck is I/O, not GPU

Will Valve Fix This?

Forza Horizon 6 is a Steam Day 1 launch title with a Steam Machine marketing tie-in. Valve has direct incentive to fix Proton compatibility. The Proton #9789 issue has 327+ community reports and is actively monitored by Valve engineers. The black screen fix in May 2026 proves they're working on it. Ray tracing and stuttering fixes are expected before Steam Machine ships.
